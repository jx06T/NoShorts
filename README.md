# NoShorts
讓連續短片消失!
###### *version-V1.0* 
---

## 下載&安裝
1.下載資料夾並解壓縮：code -> download ZIP 

2.開啟擴充頁面並打開開發者模式：擴充功能圖示(右上) -> 管理擴充功能 -> 開發人員模式

3.載入擃充資料夾：載入未封裝項目(載入解壓縮) -> 選擇下載的資料夾

4.安裝完成圖標：

![sample1](images/NoShorts128.png)

## 設定(右鍵點選圖標進入選項頁面)
  
a. 是否啟用：最上方勾選框

b. 隱藏：讓連續短片按鈕消失

c. 模糊：完全封鎖

d. 鎖定：阻止連續滾動(下方限制指最高滾動次數)
